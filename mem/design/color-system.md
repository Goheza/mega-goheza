# Goheza Color System (current)

- **Primary brand: Coral #FF5F5B** — mapped to `--primary` and `--coral` tokens in src/styles.css. Use `bg-primary`, `text-primary`, `bg-coral`, `text-coral`, `border-coral`. Drives ALL primary CTAs, active nav, links, focus rings, badges, charts.
- **Orange #FB8007 is fully retired.** Do NOT reintroduce orange anywhere. Previous orange-as-primary direction is superseded.
- **Background: #F7F4EE** warm cream + alternating warm ivory/beige/stone surfaces (`bg-surface`, `bg-surface-warm`, `bg-surface-stone`). Keep alternating section banding.
- **Dark neutral:** deep charcoal `--ink` for contrast sections and text.
- **Gradients:** `--gradient-primary` and `--gradient-coral` are both coral-to-deeper-coral (no orange stop). Use sparingly on CTAs, hero accents, dashboard highlights.
- **Status:** success=green, warning=amber, error=soft red, info=muted blue. Do not use coral for warning.
- **Secondary buttons:** transparent bg + coral border + coral text, tinted coral hover.
